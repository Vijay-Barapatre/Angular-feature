Directives are classes that add behavior to elements in Angular applications.

### Directive Types

| Type | Purpose | Example |
|------|---------|---------|
| **Component** | Directive with template | `@Component` |
| **Attribute** | Change appearance/behavior | `[ngClass]`, `[ngStyle]` |
| **Structural** | Change DOM structure | `*ngIf`, `*ngFor`, `@if`, `@for` |

## Best Patterns

1. **async pipe** - Zero boilerplate
2. **takeUntilDestroyed()** - Modern approach
3. **DestroyRef.onDestroy()** - For non-Observable cleanup

---

## Example

```typescript
private destroyRef = inject(DestroyRef);

ngOnInit() {
    this.data$.pipe(
        takeUntilDestroyed(this.destroyRef)
    ).subscribe();
}
```

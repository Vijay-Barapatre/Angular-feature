üéØ Key Concepts

@Input() - Data Down

Allows a parent component to pass data to a child component via property binding.

@Input() userName: string;

@Output() - Events Up

Allows a child component to emit events to the parent component using EventEmitter.

@Output() userClick = new EventEmitter();

Two-Way Binding

Combine @Input() and @Output() with the Change suffix for [()] syntax support.

@Input() value; @Output() valueChange;

Type Safety

Use TypeScript types and interfaces for type-safe component communication.

@Output() itemAdded = new EventEmitter<Item>();
